# 自定义分类规则系统

## 目标
创建独立的分类规则文件 `categories.yaml`，让AI根据规则自动分类账单。

## 用户自定义分类

```yaml
categories:
  车:
    - 过路费
    - 能源费用
    - 停车费
    - 车贷月供
    - 保险
    - 车载用品购物
    - 洗车美容
    - 保养维修

  宠物:
    - 主粮零食
    - 玩具
    - 医疗与洗护

  日常生活:
    - 餐饮
    - 购物
    - 医疗
    - 月供    # 话费、房租、水电等月开支
    - 其他
```

## 关键词映射规则

每个子类别配置关键词，用于自动匹配：

```yaml
keywords:
  # 车
  过路费: [高速, 过路费, ETC, 通行费]
  能源费用: [加油, 油费, 充电, 电费, 中石化, 中石油, 壳牌]
  停车费: [停车, 拍照, 车位]
  车贷月供: [车贷, 月供, 还款]
  保险: [车险, 保险, 交强险, 商业险]
  车载用品购物: [车品, 车载, 行车记录仪, 座套, 脚垫]
  洗车美容: [洗车, 打蜡, 抛光, 美容]
  保养维修: [保养, 维修, 换油, 轮胎, 刹车]

  # 宠物
  主粮零食: [猫粮, 狗粮, 猫条, 罐头, 宠物零食]
  玩具: [猫玩具, 狗玩具, 猫抓板, 逗猫棒]
  医疗与洗护: [宠物医院, 疫苗, 驱虫, 洗澡, 美容]

  # 日常生活
  餐饮: [餐, 食, 吃, 饭, 美团, 饿了么, 外卖, 奶茶, 咖啡]
  购物: [商, 超市, 淘宝, 京东, 拼多多, 服装]
  医疗: [医, 药, 诊所, 医院, 体检]
  月供: [房租, 话费, 水费, 电费, 燃气, 物业, 宽带]
```

## 文件结构

- `categories.yaml` - 分类规则（用户可自定义）
- `import.py` - 读取规则文件进行分类
- `config.yaml` - 人员配置（保持不变）

## 修改 import.py

1. 启动时加载 `categories.yaml`
2. `guess_category()` 函数改为读取规则文件
3. 按优先级匹配：车 > 宠物 > 日常生活
